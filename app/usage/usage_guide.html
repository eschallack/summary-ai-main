How to Install

Learn how to install the application by following the step-by-step instructions provided here.


Step 1

Make sure Python 3.12 and Ollama are installed on your computer

Step 2

download the code and open the

Step 3

Open up a terminal (from the top bar go to terminal > new terminal) and type the following commands in order:

python -m venv venv

venv\scripts\activate

pip install -r requirements.txt

start the program with the following terminal command:

python main.py

Basic Usage

Main Menu

You will be greeted with an options menu. Currently, the only working feature is short synopsis, which has options for single synopsis and bulk synopsis conversions. Type 1 to access the short synopsis menu


Short Synopsis Generator

1) Single Synopsis

Copy and paste in the synopsis you would like to shorten. Type the maximum character count for the synopsis (hitting enter without typing gives you the default option).
Then, Provide the model with any context about the episode/show/standalone you think might help it generate a better short synopsis -
this can be anything from the genre of the show to the synopsis of the show the episode is a part of. The formatting doesn't nessicarily matter, but i'd suggest doing it like:

genres: action and adventure, show_synopsis:the show about the thing that happened.

2) Bulk Synopsis

Metadata Format

I've included an example template for the metadata here: app\examples\short_synopsis_input_format_example.xlsx

The following fields are required:

  • id
  • title
  • synopsis

The following fields are optional, but are used to give the model additional context:

  • genre
  • keywords
  • show_synopsis
Usage

you will be asked to provide the path to the spreadsheet you wish to use. Ensure the path has no spaces, or use quotation marks around the path.

Once you provide the max character length, the bulk export will begin running.

Viewable logs are shown in the terminal, where you'll see the model iteratively attempting to create a synopsis that both makes sense and meets the character requirements. another AI Performs a quality control check on the model output to ensure it's not utter nonsense.

once complete, it will output the result next to the input spreadsheet.
